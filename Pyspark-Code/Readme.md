## PySpark Code – Medallion Architecture

This folder contains screenshots of **PySpark code** used to implement the Medallion Architecture (Bronze, Silver, Gold) for the Exam Performance Analytics project.

The code demonstrates real-world data engineering practices using Databricks and PySpark.

### Structure

- **bronze/** – Raw data ingestion logic  
- **silver/** – Data cleaning and validation logic  
- **gold/** – Business logic, KPIs, and analytics transformations  

Each subfolder contains code screenshots corresponding to its respective layer.
