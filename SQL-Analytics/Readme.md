This folder contains SQL queries used to power the **Exam Performance Analytics Dashboard** in Databricks SQL.

### Contents

- **dashboard_queries.sql**  
  SQL queries used to create dashboard visuals such as pass percentage by city, category-wise performance, attempt trends, and top students.

- **kpi_queries.sql**  
  Queries used to calculate key performance indicators (KPIs) including overall pass percentage, attendance percentage, average marks, and total present students.

- **screenshots/**  
  Contains screenshots of executed SQL queries along with their result outputs as displayed in Databricks SQL Analytics.

### Purpose

These SQL queries are executed on **Gold layer tables** generated through the Databricks Medallion Architecture.  
They enable:
- Data-driven visualizations
- KPI computation
- Interactive dashboard creation

All queries are optimized for analytics and reporting use cases.
